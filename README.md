# MercuryResource

Mercury spatial intelligence platform  -  MESSENGER imagery, surface composition, MDIS basemap, BepiColombo mission data.

Part of the [Resource Systems](https://terraresource.net) multi-planetary intelligence suite by Corgan Studio.

- **Domain:** mercuryresource.net (not yet pointed at hosting)
- **Hosting:** Cloudflare Pages
- **Engine:** Three.js (planet-scale rendering)
- **Shared library:** `public/resource-systems-core.js` (identical across all sibling sites)

## Status

**v0.1 scaffold.** Solid-colored planet sphere, shared topbar/auth/Stripe/DSN/legal pages, SEO, mobile responsive. Real terrain data and mission tracking arrive in v0.2.
